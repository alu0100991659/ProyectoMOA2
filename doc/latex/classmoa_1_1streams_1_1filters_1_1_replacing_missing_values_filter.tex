\doxysubsection{Detailed Description}
Replaces the missing values with another value according to the selected strategy. Available strategies for numerical attributes are\+:
\begin{DoxyEnumerate}
\item Nothing\+: Does nothing (doesn\textquotesingle{}t replace missing values)
\item Last\+Known\+Value\+: Replaces with the last non missing value
\item Mean\+: Replaces with mean of the processed instances so far
\item Max\+: Replaces with maximum of the processed instances so far
\item Min\+: Replaces with minimum of the processed instances so far
\item Constant\+: Replaces with a constant value (default\+: zero)
\end{DoxyEnumerate}

Available strategies for nominal attributes are\+:
\begin{DoxyEnumerate}
\item Nothing\+: Does nothing (doesn\textquotesingle{}t replace missing values)
\item Last\+Known\+Value\+: Replaces with the last non missing value
\item Mode\+: Replaces with the mode of the processed instances so far (most frequent value)
\end{DoxyEnumerate}

Beware of numerical strategies 2 to 5\+: if no previous non-\/missing values were processed, missing values will be replaced by 0.
