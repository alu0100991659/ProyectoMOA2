\doxysubsection{Detailed Description}
Budget Manager Interface to make AL Classifiers select the most beneficial instances. A Budget Manager is defined in that way that it defines a threshold internally and tests if the internal value is above or equal that threshold.

This Budget Manager assumes that a new instance is processed each time the method \textquotesingle{}is\+Above\textquotesingle{} is called and that the number of acquired instances equals the number of returned \textquotesingle{}true\textquotesingle{}s.
