\doxysubsection{Detailed Description}
Class for building and using a multinomial Naive Bayes classifier. Performs text classic bayesian prediction while making naive assumption that all inputs are independent. For more information see,~\newline
 ~\newline
 Andrew Mccallum, Kamal Nigam\+: A Comparison of Event Models for Naive Bayes Text Classification. In\+: A\+A\+A\+I-\/98 Workshop on \textquotesingle{}Learning for Text Categorization\textquotesingle{}, 1998.~\newline
 ~\newline
 The core equation for this classifier\+:~\newline
 ~\newline
 P\mbox{[}Ci$\vert$D\mbox{]} = (P\mbox{[}D$\vert$\+Ci\mbox{]} x P\mbox{[}Ci\mbox{]}) / P\mbox{[}D\mbox{]} (Bayes rule)~\newline
 ~\newline
 where Ci is class i and D is a document.~\newline
 ~\newline
 Incremental version of the algorithm.
