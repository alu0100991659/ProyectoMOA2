\doxysubsection{Detailed Description}
Defines an object to be used to split a node in an M-\/\+Tree. A node must be split when it has reached its maximum capacity and a new child node would be added to it.

The splitting consists in choosing a pair of \char`\"{}promoted\char`\"{} data objects from the children and then partition the set of children in two partitions corresponding to the two promoted data objects.
