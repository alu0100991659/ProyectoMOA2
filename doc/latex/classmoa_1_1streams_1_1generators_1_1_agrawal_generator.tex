\doxysubsection{Detailed Description}
Stream generator for Agrawal dataset. Generator described in paper\+:~\newline
 Rakesh Agrawal, Tomasz Imielinksi, and Arun Swami, \char`\"{}\+Database Mining\+: A Performance Perspective\char`\"{}, I\+E\+EE Transactions on Knowledge and Data Engineering, 5(6), December 1993. ~\newline
~\newline


Public C source code available at\+:~\newline
 \href{http://www.almaden.ibm.com/cs/projects/iis/hdb/Projects/data_mining/datasets/syndata.html}{\texttt{ http\+://www.\+almaden.\+ibm.\+com/cs/projects/iis/hdb/\+Projects/data\+\_\+mining/datasets/syndata.\+html}}~\newline
~\newline


Notes\+:~\newline
 The built in functions are based on the paper (page 924), which turn out to be functions pred20 thru pred29 in the public C implementation. Perturbation function works like C implementation rather than description in paper.
