\doxysubsection{Detailed Description}
\mbox{\hyperlink{classmoa_1_1gui_1_1visualization_1_1_abstract_graph_canvas}{Abstract\+Graph\+Canvas}} is an abstract class offering scaling functionality and the structure of the underlying Axes and Plot classes.

The functionality of the scaling is as following\+: the size of the canvas is determined by two sizes\+: the size itself and the preferred\+Size, which is used by the parental viewport to set its size. If the window is rescaled (e.\+g. by dragging the window) the preferred size does not change, which results in setting the size to the viewport size. If the y axis is zoomed, the base\+Height determines the new size and preferred size.

This class is partially based on \mbox{\hyperlink{classmoa_1_1gui_1_1visualization_1_1_graph_canvas}{Graph\+Canvas}}.
