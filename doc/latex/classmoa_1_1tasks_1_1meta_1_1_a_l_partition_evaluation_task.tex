\doxysubsection{Detailed Description}
This task extensively evaluates an active learning classifier on a stream. First, the given data set is partitioned into subsets, each leaving out a different part of the overall data. On each subset, the \mbox{\hyperlink{classmoa_1_1tasks_1_1meta_1_1_a_l_multi_param_task}{A\+L\+Multi\+Param\+Task}} is performed which individually evaluates the active learning classifier for each element of a set of parameter values. The individual evaluation is done by prequential evaluation (testing, then training with each example in sequence).
