Adaptive Model Rules for Multi\+Label problems (A\+M\+Rules\+ML), the streaming rule learning algorithm.

\begin{DoxyAuthor}{Author}

\end{DoxyAuthor}

\begin{DoxyParagraph}{Revision}

\end{DoxyParagraph}

This algorithm learns ordered and unordered rule set from data streams. Each rule detect anomalies and react to changes by pruning the rule set. This algorithm also does the detection of anomalies.
