\doxysubsection{Detailed Description}
\mbox{\hyperlink{classmoa_1_1core_1_1_fixed_length_list}{Fixed\+Length\+List}} is an extension of an Array\+List with a fixed maximum size. If an element is added that would put the list over its maximum size then the oldest element in the list is removed.
