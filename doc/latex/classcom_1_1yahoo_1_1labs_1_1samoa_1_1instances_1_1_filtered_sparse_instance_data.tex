\doxysubsection{Detailed Description}
The Class \mbox{\hyperlink{classcom_1_1yahoo_1_1labs_1_1samoa_1_1instances_1_1_filtered_sparse_instance_data}{Filtered\+Sparse\+Instance\+Data}}.

This class is an extension to the original \mbox{\hyperlink{classcom_1_1yahoo_1_1labs_1_1samoa_1_1instances_1_1_sparse_instance_data}{Sparse\+Instance\+Data}}. It has been created to be used with feature selection algorithms for data streams. In contrast to \mbox{\hyperlink{classcom_1_1yahoo_1_1labs_1_1samoa_1_1instances_1_1_sparse_instance_data}{Sparse\+Instance\+Data}} objects, missing values are represented as Na\+Ns instead of 0s (zeros). This allows learners to skip features with missing values.
