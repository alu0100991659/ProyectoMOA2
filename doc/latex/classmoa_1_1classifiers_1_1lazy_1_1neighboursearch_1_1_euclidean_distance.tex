\doxysubsection{Detailed Description}
Implementing Euclidean distance (or similarity) function.~\newline
 ~\newline
 One object defines not one distance but the data model in which the distances between objects of that data model can be computed.~\newline
 ~\newline
 Attention\+: For efficiency reasons the use of consistency checks (like are the data models of the two instances exactly the same), is low.~\newline
 ~\newline
 For more information, see\+:~\newline
 ~\newline
 Wikipedia. Euclidean distance. U\+RL \href{http://en.wikipedia.org/wiki/Euclidean_distance}{\texttt{ http\+://en.\+wikipedia.\+org/wiki/\+Euclidean\+\_\+distance}}.
