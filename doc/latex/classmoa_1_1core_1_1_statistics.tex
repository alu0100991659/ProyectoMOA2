Returns the value, {\ttfamily x}, for which the area under the Normal (Gaussian) probability density function (integrated from minus infinity to {\ttfamily x}) is equal to the argument {\ttfamily y} (assumes mean is zero, variance is one). 

For small arguments {\ttfamily 0 $<$ y $<$ exp(-\/2)}, the program computes {\ttfamily z = sqrt( -\/2.\+0 $\ast$ log(y) )}; then the approximation is {\ttfamily x = z -\/ log(z)/z -\/ (1/z) P(1/z) / Q(1/z)}. There are two rational functions P/Q, one for {\ttfamily 0 $<$ y $<$ exp(-\/32)} and the other for {\ttfamily y} up to {\ttfamily exp(-\/2)}. For larger arguments, {\ttfamily w = y -\/ 0.\+5}, and {\ttfamily x/sqrt(2pi) = w + w$\ast$$\ast$3 R(w$\ast$$\ast$2)/S(w$\ast$$\ast$2))}.
