\doxysubsection{Detailed Description}
Implements an autoencoder\+: a neural network that attempts to reconstruct the input. The autoencoder\textquotesingle{}s structure is an input layer with one neuron for each non-\/class attribute, a hidden layer of two neurons and an output layer with one neuron for each non-\/class attribute.

Partially modeled on Albert Bifet\textquotesingle{}s Perceptron.\+java, also found in the M\+OA project.

Backpropagation implementation based on Matt Mazur\textquotesingle{}s excellent tutorial on this topic\+: \href{https://mattmazur.com/2015/03/17/a-step-by-step-backpropagation-example/}{\texttt{ https\+://mattmazur.\+com/2015/03/17/a-\/step-\/by-\/step-\/backpropagation-\/example/}}
