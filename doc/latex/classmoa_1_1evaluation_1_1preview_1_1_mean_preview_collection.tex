\doxysubsection{Detailed Description}
Class that holds separate \mbox{\hyperlink{classmoa_1_1evaluation_1_1preview_1_1_preview_collection}{Preview\+Collection}}s for mean and standard deviation values. The values can be calculated from a \mbox{\hyperlink{classmoa_1_1evaluation_1_1preview_1_1_preview_collection}{Preview\+Collection}} that contains the results of multiple runs for the same parameter configurations.

On creation of a \mbox{\hyperlink{classmoa_1_1evaluation_1_1preview_1_1_mean_preview_collection}{Mean\+Preview\+Collection}}, the mean Previews and standard deviation Previews are calculated from the given \mbox{\hyperlink{classmoa_1_1evaluation_1_1preview_1_1_preview_collection}{Preview\+Collection}} by averaging the measurements for all entries over the different runs that have been performed. This results in a collection of mean Previews and a separate collection of standard deviation Previews, both containing Previews for each parameter configuration.
