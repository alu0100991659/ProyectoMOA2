M\+OA is the most popular open source framework for data stream mining, with a very active growing community (\href{http://moa.cms.waikato.ac.nz/blog/}{\texttt{ blog}}). It includes a collection of machine learning algorithms (classification, regression, clustering, outlier detection, concept drift detection and recommender systems) and tools for evaluation. Related to the W\+E\+KA project, M\+OA is also written in Java, while scaling to more demanding problems.
