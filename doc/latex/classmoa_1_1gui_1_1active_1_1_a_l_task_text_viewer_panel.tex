\doxysubsection{Detailed Description}
This panel displays text. Used to output the results of tasks. In contrast to Tast\+Text\+Viewer\+Panel, this class supports multiple curves, standard deviations and two additional views, which display a varied parameter and the relative label acquisition rate (usually called budget).
