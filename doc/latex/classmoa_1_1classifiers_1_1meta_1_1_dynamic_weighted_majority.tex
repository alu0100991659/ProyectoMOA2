\doxysubsection{Detailed Description}
Dynamic weighted majority algorithm.

Extends the Weighted Majority Algorithm to add and remove experts based on local and global accuracy.

J. Zico Kolter and Marcus A. Maloof. Dynamic weighted majority\+: An ensemble method for drifting concepts. The Journal of Machine Learning Research, 8\+:2755-\/2790, December 2007. I\+S\+SN 1532-\/4435. U\+RL \href{http://dl.acm.org/citation.cfm?id=1314498.1390333}{\texttt{ http\+://dl.\+acm.\+org/citation.\+cfm?id=1314498.\+1390333}}. 

Based on the source code provided by the author at \href{http://people.cs.georgetown.edu/~maloof/pubs/jmlr07.php}{\texttt{ http\+://people.\+cs.\+georgetown.\+edu/$\sim$maloof/pubs/jmlr07.\+php}}
