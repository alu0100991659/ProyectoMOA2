\doxysubsection{Detailed Description}
B\+L\+A\+ST (Best Last) for Heterogeneous Ensembles implemented with Fading Factors

Given a set of (heterogeneous) classifiers, B\+L\+A\+ST builds an ensemble, and determines the weights of all ensemble members based on their performance on recent observed instances. This implementation uses a window of recent instances, the size is determined by the \textquotesingle{}w\textquotesingle{} parameter. 

J. N. van Rijn, G. Holmes, B. Pfahringer, J. Vanschoren. Having a Blast\+: Meta-\/\+Learning and Heterogeneous Ensembles for Data Streams. In 2015 I\+E\+EE International Conference on Data Mining, pages 1003-\/1008. I\+E\+EE, 2015. 

Parameters\+: 


\begin{DoxyItemize}
\item -\/w \+: Window size 
\item -\/b \+: Comma-\/separated string of classifiers 
\item -\/g \+: Grace period (1 = optimal) 
\item -\/k \+: Number of active classifiers
