\doxysubsection{Detailed Description}
Instance stream which consumes instances from a Kafka topic.

Assumptions\+:
\begin{DoxyItemize}
\item The ordering of the instances in the topic is unimportant, or if it is important, it is ensured by a topic with only one partition.
\item The stream is considered ended when a record with a null value is found.
\item The serialised form of the instances is using Java\textquotesingle{}s own serialisation tools (i.\+e. \mbox{\hyperlink{classmoa_1_1streams_1_1_object_serializer}{Object\+Serializer}}).
\end{DoxyItemize}
