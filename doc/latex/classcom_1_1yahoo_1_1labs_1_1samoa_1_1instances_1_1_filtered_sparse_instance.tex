\doxysubsection{Detailed Description}
The Class \mbox{\hyperlink{classcom_1_1yahoo_1_1labs_1_1samoa_1_1instances_1_1_filtered_sparse_instance}{Filtered\+Sparse\+Instance}}.

This class is an extension to the original \mbox{\hyperlink{classcom_1_1yahoo_1_1labs_1_1samoa_1_1instances_1_1_sparse_instance}{Sparse\+Instance}}. It has been created to be used with feature selection algorithms for data streams. In contrast to \mbox{\hyperlink{classcom_1_1yahoo_1_1labs_1_1samoa_1_1instances_1_1_sparse_instance}{Sparse\+Instance}} objects, missing values are represented as Na\+Ns instead of 0s (zeros). This allows learners to skip features with missing values.
