\doxysubsection{Detailed Description}
An interface for incremental classifier models. As an extension of \mbox{\hyperlink{interfacemoa_1_1classifiers_1_1_multi_class_classifier}{Multi\+Class\+Classifier}}, these classifiers appear in the G\+UI Classification Tab. Marking them as One\+Class\+Classifiers also allows them to appear specifically when one-\/class classifiers are needed.
