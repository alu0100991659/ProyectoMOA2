~\newline


Two contexts are compatible if they follow the following rules\+:~\newline
 Rule 1\+: num classes can increase but never decrease~\newline
 Rule 2\+: num attributes can increase but never decrease~\newline
 Rule 3\+: num nominal attribute values can increase but never decrease~\newline
 Rule 4\+: attribute types must stay in the same order (although class can move; is always skipped over)~\newline
~\newline


Attribute names are free to change, but should always still represent the original attributes.
