\doxysubsection{Detailed Description}
Self Adjusting Memory (S\+AM) coupled with the k Nearest Neighbor classifier (\mbox{\hyperlink{classmoa_1_1classifiers_1_1lazy_1_1k_n_n}{k\+NN}}) .

Valid options are\+:

-\/k number of neighbours ~\newline
 -\/w max instances ~\newline
 -\/m minimum number of instances in the S\+TM ~\newline
 -\/p L\+TM size relative to max instances ~\newline
 -\/r Recalculation of the S\+TM error ~\newline
