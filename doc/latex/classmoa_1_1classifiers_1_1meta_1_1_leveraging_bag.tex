\doxysubsection{Detailed Description}
Leveraging Bagging for evolving data streams using A\+D\+W\+IN. Leveraging Bagging and Leveraging Bagging MC using Random Output Codes ( -\/o option).

See details in\+:~\newline
 Albert Bifet, Geoffrey Holmes, Bernhard Pfahringer. Leveraging Bagging for Evolving Data Streams Machine Learning and Knowledge Discovery in Databases, European Conference, E\+C\+ML P\+K\+DD\}, 2010.
