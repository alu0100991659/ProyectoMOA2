\doxysubsection{Detailed Description}
The class that splits a node into two based on the midpoint value of the dimension in which the node\textquotesingle{}s rectangle is widest. If after splitting one side is empty then it is slided towards the non-\/empty side until there is at least one point on the empty side.~\newline

Splits a node into two based on the midpoint value of the dimension in which the node\textquotesingle{}s rectangle is widest. If after splitting one side is empty then it is slided towards the non-\/empty side until there is at least one point on the empty side. The two nodes created after the whole splitting are correctly initialised. And, node.\+left and node.\+right are set appropriately. ~\newline
