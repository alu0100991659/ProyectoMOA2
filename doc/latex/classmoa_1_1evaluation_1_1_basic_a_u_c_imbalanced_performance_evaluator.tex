\doxysubsection{Detailed Description}
Performance measures designed for class imbalance problems. Only to be used for binary classification problems with unweighted instances. Class 0 -\/ majority/negative examples, class 1 -\/ minority, positive examples. A\+UC calculation as described in (but without windowing)\+: D. Brzezinski, J. Stefanowski, \char`\"{}\+Prequential A\+U\+C\+: Properties of the Area Under the R\+O\+C Curve for
\+Data Streams with Concept Drift\char`\"{}, Knowledge and Information Systems, 2017.

\begin{DoxyAuthor}{Author}
Dariusz Brzezinski (dbrzezinski at cs.\+put.\+poznan.\+pl) 
\end{DoxyAuthor}
