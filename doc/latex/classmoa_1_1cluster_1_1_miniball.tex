\doxysubsection{Detailed Description}
Java Porting of the Miniball.\+h code of {\bfseries{Bernd Gaertner}}. Look at \href{http://www.inf.ethz.ch/personal/gaertner/miniball.html}{\texttt{ http\+://www.\+inf.\+ethz.\+ch/personal/gaertner/miniball.\+html}}~\newline
 and related work at \href{http://www.inf.ethz.ch/personal/gaertner/texts/own_work/esa99_final.pdf}{\texttt{ http\+://www.\+inf.\+ethz.\+ch/personal/gaertner/texts/own\+\_\+work/esa99\+\_\+final.\+pdf}}~\newline
 for reading about the algorithm and the implementation of it.

If interested in Bounding Sphere algorithms read also published work of {\bfseries{Emo Welzl}} \char`\"{}\+Smallest enclosing disks (balls and Ellipsoid)\char`\"{} and the work of {\bfseries{Jack Ritter}} on \char`\"{}\+Efficient Bounding Spheres\char`\"{} at~\newline
 \href{http://tog.acm.org/GraphicsGems/gems/BoundSphere.c?searchterm=calc}{\texttt{ http\+://tog.\+acm.\+org/\+Graphics\+Gems/gems/\+Bound\+Sphere.\+c?searchterm=calc}}
