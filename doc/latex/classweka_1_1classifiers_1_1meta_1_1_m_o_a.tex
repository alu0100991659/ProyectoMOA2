\doxysubsection{Detailed Description}
Wrapper for \mbox{\hyperlink{classweka_1_1classifiers_1_1meta_1_1_m_o_a}{M\+OA}} classifiers.~\newline
 ~\newline
 Since \mbox{\hyperlink{classweka_1_1classifiers_1_1meta_1_1_m_o_a}{M\+OA}} doesn\textquotesingle{}t offer a mechanism to query a classifier for the types of attributes and classes it can handle, the capabilities of this wrapper are hard-\/coded\+: nominal and numeric attributes and only nominal class attributes are allowed. 

Valid options are\+: 


\begin{DoxyPre} -\/B <classname + options>
   The \mbox{\hyperlink{classweka_1_1classifiers_1_1meta_1_1_m_o_a}{MOA}} classifier to use.
   (default: moa.classifiers.DecisionStump)\end{DoxyPre}



\begin{DoxyPre} -\/D
   If set, classifier is run in debug mode and
   may output additional info to the console\end{DoxyPre}
