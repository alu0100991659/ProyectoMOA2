\doxysubsection{Detailed Description}
B\+L\+A\+ST (Best Last) for Heterogeneous Ensembles Abstract Base Class

Given a set of (heterogeneous) classifiers, B\+L\+A\+ST builds an ensemble, and determines the weights of all ensemble members based on their performance on recent observed instances. Used as Abstact Base Class for \mbox{\hyperlink{classmoa_1_1classifiers_1_1meta_1_1_heterogeneous_ensemble_blast}{Heterogeneous\+Ensemble\+Blast}} and \mbox{\hyperlink{classmoa_1_1classifiers_1_1meta_1_1_heterogeneous_ensemble_blast_fading_factors}{Heterogeneous\+Ensemble\+Blast\+Fading\+Factors}}. 

J. N. van Rijn, G. Holmes, B. Pfahringer, J. Vanschoren. Having a Blast\+: Meta-\/\+Learning and Heterogeneous Ensembles for Data Streams. In 2015 I\+E\+EE International Conference on Data Mining, pages 1003-\/1008. I\+E\+EE, 2015. 

Parameters\+: 


\begin{DoxyItemize}
\item -\/b \+: Comma-\/separated string of classifiers 
\item -\/g \+: Grace period (1 = optimal) 
\item -\/k \+: Number of active classifiers 
\end{DoxyItemize}
