\doxysubsection{Detailed Description}
Drift detection method based in \mbox{\hyperlink{classmoa_1_1classifiers_1_1core_1_1driftdetection_1_1_a_d_w_i_n}{A\+D\+W\+IN}}. A\+Daptive sliding W\+I\+Ndow is a change detector and estimator. It keeps a variable-\/length window of recently seen items, with the property that the window has the maximal length statistically consistent with the hypothesis \char`\"{}there has been no change in the average value
inside the window\char`\"{}.
