{\footnotesize\ttfamily void moa.\+options.\+Dependent\+Options\+Updater.\+refresh\+Varied\+Param\+Name\+Option (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})}

Refresh the provided choices of an \mbox{\hyperlink{classmoa_1_1options_1_1_editable_multi_choice_option}{Editable\+Multi\+Choice\+Option}} every time a \mbox{\hyperlink{classmoa_1_1options_1_1_class_option}{Class\+Option}} (the prequential evaluation task) is changed. This method checks if the chosen task actually changed, before updating the Multi\+Choice\+Option. ~\newline
 The method looks for available options in the learner option of the prequential evaluation task and recursively in all of its Class\+Options. ~\newline
 Only Int and Float options are shown in the Multi\+Choice\+Option, because only numeric parameters should be variable. ~\newline
 If one of the options is named \char`\"{}budget\char`\"{} or its name contains the word \char`\"{}budget\char`\"{}, it is selected as the default option.
