\doxysubsection{Detailed Description}
Manages setting the look and feel. Uses the \mbox{\hyperlink{classmoa_1_1gui_1_1_look_and_feel_aa839e8dd6c0bac480af179e00d609cf2}{K\+E\+Y\+\_\+\+L\+O\+O\+K\+A\+N\+D\+F\+E\+EL}} property from \mbox{\hyperlink{classmoa_1_1gui_1_1_g_u_i_defaults}{G\+U\+I\+Defaults}} to determine what Look\textquotesingle{}n\textquotesingle{}Feel to use. See examples in the G\+U\+I.\+props file for more details.
