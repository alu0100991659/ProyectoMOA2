\doxysubsection{Detailed Description}
This is an interface for classes that are to be given along with every data point inserted in the tree. The tree has to inform the implementation of this {\ttfamily \mbox{\hyperlink{interfacemoa_1_1clusterers_1_1clustree_1_1util_1_1_budget}{Budget}}} interface of every operation it does, and ask at the places where it can stop prematurely if it should. This models the arrival of a new data point in the stream, before there was enough time to insert the current one.
