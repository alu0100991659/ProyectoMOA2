Detailed Description

An interface for incremental classifier models. As an extension of MultiClassClassifier, these classifiers appear in the GUI Classification Tab. Marking them as OneClassClassifiers also allows them to appear specifically when one-class classifiers are needed.

Author
Richard Hugh Moulton
