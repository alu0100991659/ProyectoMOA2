Detailed Description

Class implementing the Cobweb and Classit clustering algorithms. See: http://en.wikipedia.org/wiki/Cobweb_%28clustering%29

Citation: D. Fisher (1987). Knowledge acquisition via incremental conceptual clustering. Machine Learning. 2(2):139-172.
