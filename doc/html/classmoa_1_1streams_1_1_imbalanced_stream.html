Detailed Description

Imbalanced Stream.

This is a meta-generator that produces class imbalance in a stream. Only two parameters are required to be set:

  • The original stream
  • The ratio (proportion) of each class in the stream.

The second parameter determines the ratio of each class in the output stream. The ratio of each class should be provided as a real number between 0.0 and 1.0, each being followed by a semicolon, and their sum should add up to 1.0. The default value of 0.9;0.1 stands for an output stream where approximately 90% of the instances belonging to the first class while the remainder 10% would belong to the secondary class.
