Detailed Description

Instance stream which consumes instances from a Kafka topic.

Assumptions:

  • The ordering of the instances in the topic is unimportant, or if it is important, it is ensured by a topic with only one partition.
  • The stream is considered ended when a record with a null value is found.
  • The serialised form of the instances is using Java's own serialisation tools (i.e. ObjectSerializer).
