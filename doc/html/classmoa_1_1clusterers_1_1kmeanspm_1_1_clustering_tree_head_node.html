Detailed Description

Provides a ClusteringTreeNode with an extended nearest neighbor search in the root.

Citation: Hendrik Fichtenberger, Marc Gill√©, Melanie Schmidt, Chris Schwiegelshohn, Christian Sohler: BICO: BIRCH Meets Coresets for k-Means Clustering. ESA 2013: 481-492 (2013) http://ls2-www.cs.tu-dortmund.de/bico/
