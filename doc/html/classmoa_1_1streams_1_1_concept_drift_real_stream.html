Detailed Description

Stream generator that adds concept drift to examples in a stream with different classes and attributes. Example: real datasets.

Example:

ConceptDriftRealStream -s (ArffFileStream -f covtype.arff) \
-d (ConceptDriftRealStream -s (ArffFileStream -f PokerOrig.arff) <br/> -d (ArffFileStream -f elec.arff) -w 5000 -p 1000000 ) -w 5000 -p 581012

s : Stream
d : Concept drift Stream
p : Central position of concept drift change
w : Width of concept drift change
