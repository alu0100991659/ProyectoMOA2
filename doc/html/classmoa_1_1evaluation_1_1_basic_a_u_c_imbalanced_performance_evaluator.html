Detailed Description

Performance measures designed for class imbalance problems. Only to be used for binary classification problems with unweighted instances. Class 0 - majority/negative examples, class 1 - minority, positive examples. AUC calculation as described in (but without windowing): D. Brzezinski, J. Stefanowski, "Prequential AUC: Properties of the Area Under the ROC Curve for Data Streams with Concept Drift", Knowledge and Information Systems, 2017.

Author
Dariusz Brzezinski (dbrzezinski at cs.put.poznan.pl)
